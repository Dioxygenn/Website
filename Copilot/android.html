Téléchargement de Paivaa

Pour les dispositifs mobils Android et apparantés


Première étape

Après avoir consulté les caractéritiques techniques de votre télephone il vous faudra téléchargé l'application. Pour ce faire vous devez appuyez sur le boutton ci-dessous, ce boutton vous guidera jusqu'a une page de téléchargement Copilot




Seconde étape

Vous avez maintenant téléchargé un document .apk ou .aab, dans les deux cas la procédure est la même. Appuyez sur télécharger puis autoriser (cela peut-etre different d'un appareil à l'autre) néanmois cela reste globalement le même. Il se peut que vous aillez une alerte vous disant que l'application est peut-être dangeureuse, cela peut venir du fait du certificat, certificat qui prouve a Google que notre application est sans risque, vous pouvez vous même consulter le certificat ci-dessous.




Dernière étape

Maintenant que Paivaa est sur votre télephone, nous vous enverons des mails pour vous prévenir des mises à jour. N'oubliez pas de faire des retours, nous vous fournirons le lien d'un questionaire unique que vous pourrez remplir quand bon vous semple et le nombre de fois que vous voulez.
